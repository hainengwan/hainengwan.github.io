智能表单

  • 新的表单类型，不建议在网页中使用
  • 可以针对需求使用特定的只能表单类型

    • <input type="datetime">兼容性不佳，可以用<input type="date">不能在PC上用，样式无法自定义

    • <input type="range" min="1" max="10" step="5" value="5"> 定义带有 slider 控件的数字字段。value：默认 step：单步选择

    • <input type="text" required>必填选项，常用于移动端，无法设置样式
    • <input type="tel" placeholder="(555)555-5555" pattern="">本身不具备校验

SVG

  • Scalable Vector Graphics：可缩放矢量图形

  • 位图就是由一堆像素点组成，由有颜色的像素

  • 矢量图记录的是线，路径，填充色，任意缩放，图片不会失真，基于可扩展标记语言

  • 文档标题<?xml version="1.0" encoding="utf-8"……> 为xml文档

  • 优点：体积小（数据量少），质量高（放大无锯齿），效果好，可控程度高

  • 定义 线性渐变 stop circle 定义了一个圆 定义了url(#id名)

  • 可以用CSS填充颜色

  • 图标字体

  • 以文件的方式使用 <iframe src="demo.svg" frameholder="0"></iframe> 推荐使用，不能通过CSS控制样式

  • <object data="demo.svg" type=""></object> 在标准中有定义

  • <embed src="demo.svg" type=""> 在标准中未定义，但浏览器都支持

  • <svg data-src="demo.src"></svg> 浏览器不支持，学了异步之后

  • 引入SVG文件后用CSS控制颜色

  • 用于高清的logo图，图标

  • 可以在网页中使用别人做好的SVG；可以对SVG文件做出基本调整；会用样式修饰SVG

  • iframe .svg本身是个文档，可以通过iframe载入进来，无法改别人的样式

svg文件的基本使用

  • https的默认端口是443
  • http的默认端口是800
  • ajax直接可以修改样式

h5下

基础API提升

  • document.getElement……appendChild……都属于基础API

  • 新选择器

    • document.querySelector(selector)——返回第一个满足选择器条件的元素，返回一个dom对象
      • 快捷键 qs=querySelector
      • ae=addEventListenter
    • docuemnt.querySelectorAll(“.item”)——返回所有满足该条件的元素，返回元素类型为dom对象的数组
    • $（”.item”）——返回一个jQuery对象（dom元素的数组）
    • 本质上jQuery的方式和querySelector的方式都是获取DOM数组，只不过jQuery会多一些其它成员
    • DOM数组的每一个成员注册事件不能像jQuery一样直接注册，必须分别给每个元素添加
    • h5就是将我们经常需要的操作又包装一层，原生API

  • Element.classList

    • 新H5中DOM对象多了一个classList属性，是一个数组

    • a标签有个默认跳转

    • this.classList.add()——添加一个新的类名

    • this.classList.remove()——删除一个类名

    • this.classList.contains()——判断是否包含一个指定的类名

    • this.classList.toggle()——切换一个class

      • toggle函数的第二个参数true为添加，false删除

  • 访问历史API

    • 界面上的所有JS操作不会被浏览器记住，就无法回到之前的状态

    • go函数可以传入一个地址，不传默认刷新

    • window.history.forward()

    • auto file name(插件plugin名字)

    • 不建议用*号把默认样式去掉，效率低

    • 默认样式有必要，只不过为了统一设置

    • 当我们在伪造的访问历史中前进或后退时会执行一个popstate事件。

    • 通过JS可以加入一个访问状态

    • history.pushState(放入历史中的状态数据，设置title（现在浏览器不支持）)

    • window.location可以拿到当前网页中跟地址相关的信息（location是个对象） hash是锚点值

    • decodeURI作用就是从URL编码转换到之前的状态

    • encode反作用

    • new Date() 获取的是一个时间对象

    • toLocalTimeString 转换成我们能认识的时间字符串

    • toDateTimeString 转换成我们能认识的日期字符串

  • class

    • 之前

    • 通过dom对象的classList，className是字符串的形式，classList是数组的形式

      • 通过classList封装了一些针对增加、删除、切换的方法——add(“newclass”)； remove(“oldclass”)； toggle(“cl”) 存在则删除，不存在添加；contains()是否包含
